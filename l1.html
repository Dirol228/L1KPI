US6 — Видалення акаунта (Remove Account)
User selection for removing
Поведінка: кнопка Remove активна тільки після вибору користувача; перед видаленням показати підтвердження.
Acceptance: видалення лише після вибору та підтвердження; вікно не змінює розміри.
US10 — Вибір звіту для редагування (Report selection)
Report selection for editing
Поведінка: Open відкриває вікно редагування тільки якщо Id обрано, інакше показати повідомлення "Report Id is not selected".
Acceptance: список порожній за замовчуванням; після відкриття відображає всі доступні звіти.
